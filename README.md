<h2 align="center" id="1">스터디 진행 방법</h2>

1. 진행할 분량을 정한다.
2. 매주 스터디를 진행할 호스트와 부호스트를 2명씩 랜덤으로 뽑는다.
3. 각자 정해진 분량을 미리 읽어온다.
4. 호스트는 해당 챕터 학습 진행의 책임을 맡는다.
   - 진행방법은 호스트의 자율에 맡긴다.
5. 스터디 진행 중 기록한 이슈(질문 등)를 공유한다.
